PL PREDICT THE SCORELINE HELP

 

 

Welcome to PL Predict The Scoreline Help.

You can write about any query that you may have from the Write To Us option in your app. You can access it by opening the left drawer (Swipe from left to right on the screen) and click Write To Us after you have logged in. We will try to get to you ASAP.

The UI of the app is very self-explanatory. You can navigate to all the screens from your profile homepage, by opening the drawer navigation toolbar (swipe your screen left to right) and also opening the menu (by either clicking the menu button from your device or by clicking the 3 dots on right corner of the action bar).

Here are some FAQ for you to introduce you to the various aspects of PL Predict The Scoreline.



MATCHDAY:

A matchday is simply one week of Premiere League. On an average a Matchday will consist of 10 matches, spread over the course of the week. On every matchday, the user can predict in the matches listed on that week. If he/she predict in all the matches, the user gets a bonus of +100 points. Failure to predict in atleast 50% of the matches will result in a penalty. In matchdays that have odd number of matches (say n) , the user needs to predict in atleast (n+1/2) number of matches. For deduction and bonus details, refer to the heading "Scoring System".

NET POINTS vs PURE POINTS:

Pure points are the points the user has acheived from predicting in the matches.
Net points is the points after the user is awarded with the Participation bonus or charged with the cutoff penalty.

(User recieves a participation bonus of 100 points when he/she predicts in all the matches from a particular matchday. If the user fails to predict in atleast 50% of the matches in a matchday, he/she incurs a penalty of 20 points for each match he predicts less than the cutoff. Read the Prediction Scoring System for more info.)

FRIENDS:

User can send friend request to any other user on the community. The second user needs to accept the request for the two users to be friends.

Friends can view each other's profile, their group history, points history, prediction history and also include them when creating groups.


GROUPS:

Groups can be created by the "CREATE GROUP" button on the Group Page. The user who creates the group can choose which friends to add in the group.

In the beta version, users can not add members to the group after it's inception.

In groups, the users are ranked only based on the pure points achieved by the group members on or after the matchdays in which the group was created. This means that if a group was created on (say) Matchday 4, the members are ranked based on the pure points accumulated by them on Matchday 4 and onwards up to Matchday 38. The pure points acheived from Matchday 1 to Matchday 3 is not considered in the evaluation of group points. This ensures an even playing field for all the members of the group.

 

 

SCORING SYSTEM:

All prediction lines are closed 2 minutes after the scheduled start of play. No prediction will be accepted after this time. Users are advised to save their predictions long before the match starts to avoid server complications.

The scores will be updated and users can see their performance after the end of that matchday's games.

 

AWARDS

A. For correctly predicting a winner or draw : +20 points

B. For perfectly predicting a scoreline (correct home and correct away score) : +50 points

C. For correctly predicting one of the side's score: +20 points (if not eligible for Award B)

D. For correctly predicting a scorer: +20 points

E. The user will get +20 points for each correct scorer you predict.

F. The user will get +20 points for each goal scored by the player you predicted.

G. In case the user predicts 0 in a scoreline, the user can choose either "PASS" or "NO SCORERS" options. If the user chooses "NO SCORERS", he/she gets +20 if there are actually no scorers for that team in the match (own goals for either team is not considered), -10 if there are goals for that team in the match. The user gets no awards, nor deductions if the user chose "Pass".

H. If the user predicts in ALL the listed matches in a matchday, he gets a +100 points bonus.


 

DEDUCTIONS

A. For getting the score of either side's scoreline off by more than 2 goals, the user gets a -10 points deduction for every defecient goal more than 2.

(For example if an user predicts that Arsenal-Chelsea scoreline would be 1-1, but it ends up being 4-4, then the user will suffer a loss of -20 points. The breakdown is as follows: -10 for missing Arsenal's score by more than 2 goals. -10 for missing Chelsea's score by more than 2 goals. Note than if the score was 5-5, then the losses will be additive and user will suffer a loss of -40 points.)

 

B. The user will suffer a loss of 10 points for each wrong scorer he predicts.

C. If the user fails to predict in the minimum number of matches that he requires to predict in that matchday (50% of the total matches on that week), the user suffers a -20 point deficit for each missing match. (For example, a typical week of PL will consist of 10 matches. The user should predict in atleast 5, to avoid a penalty. If the user predicts in just 3 matches, he suffers a loss of 20 X (5-3) = 40 points)

 

 

VIEWING STATS

There are 3 types of Stats an user can view in PLPREDICTOR app:

A. PERSONAL STATS

The user can view the detailed history of all the points he/she has attained.

The Matchday Stat window displays all the points (pure and net) scored on every single matchday.

Clicking on any of the matchday will open up the detailed breakdown of the points scored on that matchday, ie the points on each match as well as bonus and deduction.

Clicking on a particualar match will show the user a breakdown for the scores recieved on that match.

 

B. GLOBAL STATS

The user can also view global statistics of all the users in PL PREDICTOR.

They can view the leaderboards in terms of net points, they can view the ranking of pure-points recieved on a particular matchday and even the ranking of points recieved on a particular match.


C. PL STATS

Here, the user can view the current league standings, the results of previous matches and team stats of all the teams.
